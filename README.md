# gulp-startup

切图必备, 自动编译less/postcss/scss, 自动添加css3前缀, 自动刷新浏览器

```bash
# 安装依赖
npm install  or  yarn

# less 模式
npm run less

### 或者

gulp start_less

# scss 模式
npm run scss

### 或者

gulp start_scss

# postcss-salad 模式

npm run postcss

### 或者

gulp start_postcss
```

打开 http://localhost:9092

自己建立切图文件夹后, 记得修改`gulpfile.js`的`basedir`
